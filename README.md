# Windows Hinting

Initial prototype of a fast hinting program for windows 

Ctrl + T -> Display hints on taskbar
Ctrl + H -> Display hints on foreground window
Ctrl + R -> Switch action to Right click
Ctrl + L -> Switch action to Left click
Ctrl + D -> Switch action to Double click

May be activated via NamedPipe for improved accessibility for e.g popup menus - works very well with Talon Voice and stream deck.
